Class: Window

Window(id, args)

new Window(id, args)

You can look at the "settings.html" in the root folder of the project to get a rough idea of how its done. In general, its best to start with the style information. Write it as if you would write an actual website, only do not use <head>, <body> etc. You can and should include <style> Information. When using subpages, its recommended that you dynamically change the window body or the relevant parts with JS, the windowClass settingswindow does have subpages, but it can lead to problems.
Parameters:
Name Type Description
id int The id of the Window. Please use the addWindow() function.
args object Same thing, please use addWindow() and refer to its documentation.
