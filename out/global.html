errorMessage(parent, layout, title, text, functions) → {object}

Generates and renders an error message using a similar system to VB MsgBox. The type of error message can be defined via a seven-bit integer as such:
ValuesMeaning
xxxx000OK
xxxx001CANCEL OK
xxxx010ABORT RETRY IGNORE
xxxx011YES NO CANCEL
xxxx100YES NO
xxxx101RETRY CANCEL
xxx0xxxParent not blocked
xxx1xxxParent blocked
x00xxxxCRITICAL
x01xxxxQUESTION
x10xxxxEXCLAMATION
x11xxxxINFORMATION
0xxxxxxVisually parented to window
1xxxxxxVisually not tied to window
Parameters:
Name Type Description
parent object DOM Object that the error message will be bound to.
layout number Integer defining the type of error message, refer to function description
title string Title of the error message
text string The error message text
functions object Array of functions that determine the behaviour of the buttons
