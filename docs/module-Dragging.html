(static) dragElement(elmnt)

Adds the necessary event handlers to a div to allow it to be dragged around. If the container div does not have a child div with the id of the parent with "header" appended to it, it can be dragged from everywhere inside the div, if the header is present, only the header will trigger the event

Parameters:
Name Type Description
elmnt DOMElement:div

The element to add the dragging function to
